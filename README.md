
# SuperMart 🛒

A C++ console application simulating a retail inventory management system.

## Features

- 📦 View full product inventory with details (name, model, price, rating, quantity)
- 🔍 Search for products by name
- 📊 Sort inventory by Price, Rating, or Quantity
- ⚠️ Automatic LOW STOCK flag for items below threshold

## How to Run

```bash
g++ main.cpp -o supermart
./supermart
```
